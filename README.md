#  EventHub - Event Ticket Booking System

EventHub is a simple and responsive event ticket booking website built using **HTML, CSS, and JavaScript**.

This project was developed as part of my **Front-End Development Internship at Sysslan IT Solutions**.

---

##  Project Overview

EventHub allows users to:

- Browse upcoming events
- View event details
- Book tickets online
- Fill out a booking form with validation
- Use the website on desktop and mobile devices

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript

1. Download or clone this repository.
2. Open the `level5/index.html` file in your web browser.

No installation is required.
